# Tic Tac Toe - Python Tkinter

A classic Tic-Tac-Toe game for two players, implemented in Python using the built-in tkinter library for the graphical interface.

## How to Play
- Players take turns clicking on empty squares to place their mark (X or O).
- The game announces the winner when three marks align horizontally, vertically, or diagonally.
- If all 9 squares are filled without a winner, the game ends in a tie.
- Press the "restart" button to start a new game at any time.

## Requirements
- Python 3.x (tkinter comes pre-installed with most Python installations)

## How to Run
Save the script as `tictactoe.py` and run it with:
```bash
python tictactoe.py
